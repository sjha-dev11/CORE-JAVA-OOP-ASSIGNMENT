## 📖 Overview
This repository contains a robust, console-based Java application designed to manage Student Information, Employee Details, and Product Billing. Developed for a training institute, the system heavily relies on fundamental **Object-Oriented Programming principles** to ensure modularity, scalability, and code reusability. 

The project strictly fulfills its functional requirements without relying on any external frameworks—utilizing pure core Java structures and concepts.

## 🚀 Key Concepts Demonstrated

The application acts as a comprehensive technical demonstration of the following core Java concepts:

* **Object-Oriented Architecture (`Classes` & `Objects`)**: Structured modeling mapping real-world instances using modular properties.
* **Encapsulation (`private`, Getters, & Setters)**: Protecting sensitive data (e.g., product pricing) behind custom accessors.
* **Inheritance (`extends`)**: Demonstrating a generic `Student` parent class and extending functionality securely into a specialized `CollegeStudent` subclass.
* **Method Overloading (Polymorphism)**: Dynamic compute engines in the Billing class that handle varying combinations of items smoothly.
* **Constructor Initialization & `this` Keyword**: Clean object instantiation ensuring contextual assignment for property values dynamically.
* **Shared Memory State (`static`)**: Managing real-time overall counters and shared strings seamlessly across independently created object instances.
* **Data Organizing (`Arrays` & Loops)**: Utilizing primitive arrays effectively and processing elements functionally using logical iterative boundaries.
* **String Manipulations**: Using native methods to parse strings uniformly across the display output (e.g., `length()`, `toLowerCase()`, `toUpperCase()`).

## 🛠️ Instructions to Run

To run this application locally, you will need the Java Development Kit (JDK) installed on your system.

1. **Clone or Download** the project repository.
2. **Open a terminal** and navigate explicitly to the project's source directory:
   ```bash
   cd core-java-oop-assignment/src
   ```
3. **Compile the program:**
   ```bash
   javac *.java
   ```
4. **Execute the Application:**
   ```bash
   java Main
   ```
